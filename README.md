# ZaoApp

> 从一句话，到一个产品<br>
> Turn a single sentence into a usable mini app inside ZaoApp.

---

## 🚀 项目简介

ZaoApp 是一个 **AI 应用生成平台**。

用户只需用自然语言描述需求，系统即可自动生成一个可运行在 ZaoApp 内部的小应用，无需编程。

### 示例

输入：

> 做一个每日习惯打卡应用

输出：

- 今日习惯列表
- 完成/未完成状态
- 添加习惯功能
- 本地数据存储
- 可在 ZaoApp 内直接运行

---

## 🧠 核心理念

> 软件不再被“开发”，而是被“描述”生成。
